# corex-zombies

> Server-coordinated zombie AI with eight distinct zombie types.

Part of the [COREX Framework](https://github.com/ABUGIZA/COREX-Framework).

## Install

Drop the `corex-zombies` folder into:
```
server-file/resources/[corex]/corex-zombies/
```

Make sure it loads after `corex-core`:
```cfg
ensure corex-core
ensure corex-zombies
```

## Update

Download the latest release ZIP from the **Releases** tab and replace the folder.
